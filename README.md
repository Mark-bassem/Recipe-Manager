# The Recipe Manager

A professional web app for discovering recipes based on your ingredients.

## Features
- Add ingredients, search recipes (shows matches/missing).
- Add custom recipes (persistent).
- Favorites with localStorage.
- Recipe details & share links.
- Responsive design, notifications.

## Usage
1. Open `main.html`.
2. Add ingredients (Enter/Add).
3. Click Search – see recipes!
4. Favorite ❤️, view details, share.

## Files
- `main.html`: Home/search.
- `favourites.html`: Saved favorites.
- `add-recipe.html`: Add new.
- `recipe.html`: Details.
- `script.js`: Professional module.
- `style.css`: Modern styling.
- Images in root.

Fully static – no server needed!
